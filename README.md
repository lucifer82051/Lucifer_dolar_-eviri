# Lucifer USD ↔ TRY Çevirici

**Sahibi:** Lucifer  
**Proje Türü:** Döviz Çevirici / Frontend  
**Kur:** 1 USD = 43,13 TL (sabit)  
**Tarih:** 2038 Cyberpunk Edition

---

## Açıklama

Bu proje, Amerikan Doları ve Türk Lirası arasında **anlık ve doğru bir şekilde** dönüşüm yapar.  
Kur **sabit olarak 43,13 TL** alınmıştır ve hesaplamalar buna göre gerçekleşir.  

Kullanıcılar:
- USD → TRY çevirisi
- TRY → USD çevirisi

yapabilirler. Hesaplama motoru basittir ama **%100 güvenilirdir**.  

Proje tasarımı **2038 yılı cyberpunk stilinde** olup, modern bir görsellik sağlar.

---

## Kullanım

1. `index.html` dosyasını tarayıcıda açın.
2. Üst alana USD miktarını girin → alt alanda TL sonucu görünür.
3. Alt alana TL miktarını girin → üst alanda USD sonucu görünür.
4. Orta swap butonuyla yön değiştirebilirsiniz.
